== Description ==
This plugin provides deployers with a smarter way to annotate and sort
reports. It's aim is to speed up report processing during situations that
generate volumes of data, reduce volunteer burnout and help other responders
to spot emerging problems much faster

== Installation ==
1. Copy the entire /frontlinesms/ directory into your /plugins/ directory.
2. Activate the plugin.
